# NAS-QNAP-ICINGA2

Provides checks for QNAP NAS Systems. 



## Capacity Check
Checks the Capacity

## Volume Check
Checks all Volumen on inconsitencies

## HDD Temp Check
Checks the Temperature of all HDDs

## Sys Temp Check
Checks the Temperature of the System

## HDD Smart Check
Checks the SMART Status of all HDDs
